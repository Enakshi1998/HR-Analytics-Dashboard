# HR-Analytics-Dashboard

## Project Overview
This project analyzes employee attrition trends using Power BI.

## Tools Used
- Power BI
- Excel
- Data Analysis

## Key Insights
- Attrition by Department
- Attrition by Age Group
- Attrition by Salary Slab
- Gender-wise Attrition Analysis
